### Core Concepts
- **Responsive Web Design (RWD)**: Fluid grids, flexible images, and CSS media queries enable a single layout to dynamically adjust to any screen size.
- **Adaptive Web Design (AWD)**: Multiple fixed layouts are pre-built for specific screen widths (e.g., 320px, 480px, 768px, 1024px, 1200px+), and the server serves the appropriate version based on device detection.
- **Mobile-First Design**: Prioritizing the mobile experience and then scaling up for larger screens, reflecting user behavior patterns.
- **Device-Specific Optimization**: Tailoring content, controls, and performance (load times, image resolution) to the capabilities and constraints of each device category.
- **Content Control and Predictability**: AWD gives designers precise control over content presentation per device; RWD offers maintainability with one codebase.
